nginxжҳҜд»Җд№Ҳ

Nginx жҳҜдёҖж¬ҫиҪ»йҮҸзә§зҡ„ ==HTTP жңҚеҠЎеҷЁ==пјҢйҮҮз”Ё==дәӢд»¶й©ұеҠЁзҡ„ејӮжӯҘйқһйҳ»еЎһ==еӨ„зҗҶж–№ејҸжЎҶжһ¶пјҢиҝҷи®©е…¶е…·жңүжһҒеҘҪзҡ„ IO жҖ§иғҪпјҢж—¶еёёз”ЁдәҺжңҚеҠЎз«Ҝзҡ„==еҸҚеҗ‘д»ЈзҗҶ==е’Ң==иҙҹиҪҪеқҮиЎЎ==

webжңҚеҠЎеҷЁпјҡиҙҹиҙЈеӨ„зҗҶе’Ңе“Қеә”з”ЁжҲ·иҜ·жұӮпјҢдёҖиҲ¬д№ҹз§°дёәhttpжңҚеҠЎеҷЁпјҢеҰӮ ApacheгҖҒIISгҖҒNginx

еә”з”ЁжңҚеҠЎеҷЁпјҡеӯҳж”ҫе’ҢиҝҗиЎҢзі»з»ҹзЁӢеәҸзҡ„жңҚеҠЎеҷЁпјҢиҙҹиҙЈеӨ„зҗҶзЁӢеәҸдёӯзҡ„дёҡеҠЎйҖ»иҫ‘пјҢеҰӮ TomcatгҖҒWeblogicгҖҒJbossпјҲзҺ°еңЁеӨ§еӨҡж•°еә”з”ЁжңҚеҠЎеҷЁд№ҹеҢ…еҗ«дәҶwebжңҚеҠЎеҷЁзҡ„еҠҹиғҪпјү

nginxе’ҢtomcatеҢәеҲ«

Nginx жҳҜHTTP ServerпјҢTomcat еҲҷжҳҜдёҖдёӘApplication ServeпјҢе…¶жң¬иҙЁдёҠе®ғжҳҜдёҖдёӘServlet/JSPеә”з”Ёзҡ„е®№еҷЁ

HTTP Server е…іеҝғзҡ„жҳҜ HTTP еҚҸи®®еұӮйқўзҡ„дј иҫ“е’Ңи®ҝй—®жҺ§еҲ¶пјҢжүҖд»ҘеңЁ Apache/Nginx дёҠдҪ еҸҜд»ҘзңӢеҲ°д»ЈзҗҶгҖҒиҙҹиҪҪеқҮиЎЎзӯүеҠҹиғҪгҖӮе®ўжҲ·з«ҜйҖҡиҝҮ HTTP Server и®ҝй—®жңҚеҠЎеҷЁдёҠеӯҳеӮЁзҡ„иө„жәҗпјҲHTML ж–Үд»¶гҖҒеӣҫзүҮж–Үд»¶зӯүпјүгҖӮйҖҡиҝҮ CGI жҠҖжңҜпјҢд№ҹеҸҜд»Ҙе°ҶеӨ„зҗҶиҝҮзҡ„еҶ…е®№йҖҡиҝҮ HTTP Server еҲҶеҸ‘пјҢдҪҶжҳҜдёҖдёӘ HTTP Server е§Ӣз»ҲеҸӘжҳҜжҠҠжңҚеҠЎеҷЁдёҠзҡ„ж–Үд»¶еҰӮе®һзҡ„йҖҡиҝҮ HTTP еҚҸи®®дј иҫ“з»ҷе®ўжҲ·з«ҜгҖӮ

еә”з”ЁжңҚеҠЎеҷЁпјҢйЎҫеҗҚжҖқд№үжҳҜдёҖдёӘеә”з”Ёзҡ„е®№еҷЁгҖӮе®ғйҰ–е…ҲйңҖиҰҒж”ҜжҢҒејҖеҸ‘иҜӯиЁҖзҡ„ RuntimeзҺҜеўғпјҲеҜ№дәҺ Tomcat жқҘиҜҙпјҢе°ұжҳҜ JavaпјүпјҢдҝқиҜҒеә”з”ЁиғҪеӨҹеңЁеә”з”ЁжңҚеҠЎеҷЁдёҠжӯЈеёёиҝҗиЎҢгҖӮе…¶ж¬ЎпјҢйңҖиҰҒж”ҜжҢҒеә”з”Ёзӣёе…ізҡ„и§„иҢғпјҢдҫӢеҰӮзұ»еә“гҖҒе®үе…Ёж–№йқўзҡ„зү№жҖ§гҖӮеҜ№дәҺ Tomcat жқҘиҜҙпјҢе°ұжҳҜйңҖиҰҒжҸҗдҫӣ JSP/Sevlet иҝҗиЎҢйңҖиҰҒзҡ„ж ҮеҮҶзұ»еә“гҖҒInterface зӯүгҖӮдёәдәҶж–№дҫҝпјҢеә”з”ЁжңҚеҠЎеҷЁеҫҖеҫҖд№ҹдјҡйӣҶжҲҗ HTTP Server зҡ„еҠҹиғҪпјҢдҪҶжҳҜдёҚеҰӮдё“дёҡзҡ„ HTTP Server йӮЈд№ҲејәеӨ§пјҢжүҖд»Ҙеә”з”ЁжңҚеҠЎеҷЁеҫҖеҫҖжҳҜиҝҗиЎҢеңЁ HTTP Server зҡ„иғҢеҗҺпјҢжү§иЎҢеә”з”ЁпјҢе°ҶеҠЁжҖҒзҡ„еҶ…е®№иҪ¬еҢ–дёәйқҷжҖҒзҡ„еҶ…е®№д№ӢеҗҺпјҢйҖҡиҝҮ HTTP Server еҲҶеҸ‘еҲ°е®ўжҲ·з«ҜгҖӮ

2пјүеҲӣе»әnginxй…ҚзҪ®ж–Үд»¶

е…ҲеҲӣе»әдёҖдёӘдёҙж—¶е®№еҷЁпјҢз„¶еҗҺд»Һе®№еҷЁдёӯеӨҚеҲ¶ default.confеҮәжқҘпјҢеҸҜд»ҘиҮӘиЎҢдҝ®ж”№default.confпјҢиҮӘе®ҡд№үй…ҚзҪ®йЎ№гҖӮ
з„¶еҗҺеҶҚеҲӣе»әжӯЈејҸдҪҝз”Ёзҡ„ nginx е®№еҷЁе№¶жҢӮиҪҪгҖӮ
е°Ҷе®№еҷЁж–Үд»¶жӢ·иҙқеҲ°дё»жңәдёӯе‘Ҫд»Өпјҡdocker cp е®№еҷЁid:/е®№еҷЁж–Үд»¶и·Ҝеҫ„/ж–Үд»¶еҗҚ /дё»жңәж–Үд»¶и·Ҝеҫ„

docekrдёӯnginxзҡ„й»ҳи®Өи·Ҝеҫ„дҪҚзҪ®

  • confпјҡ/etc/nginx/nginx.conf

  • htmlпјҡ/usr/share/nginx/html

    зҪ‘з«ҷж–Үд»¶еӯҳж”ҫзҡ„ең°ж–№, й»ҳи®ӨеҸӘжңүNginxж¬ўиҝҺйЎөйқў, еҸҜд»ҘйҖҡиҝҮж”№еҸҳNginxй…ҚзҪ®ж–Үд»¶зҡ„ж–№ејҸжқҘдҝ®ж”№иҝҷдёӘдҪҚзҪ®

  • logпјҡ/var/log/nginx

    access.logпјҡжҜҸдёҖдёӘи®ҝй—®иҜ·жұӮйғҪдјҡй»ҳи®Өи®°еҪ•еңЁиҝҷдёӘж–Үд»¶дёӯ

    error.logпјҡ д»»дҪ•Nginxзҡ„й”ҷиҜҜдҝЎжҒҜйғҪдјҡи®°еҪ•еҲ°иҝҷдёӘж–Үд»¶дёӯ

nginxе®№еҷЁ/etc/nginx/conf.dж–Үд»¶еӨ№еҶ…пјҢиҮӘеёҰdefault.confй…ҚзҪ®ж–Үд»¶гҖӮе’ҢredisдёҚдёҖж ·пјҢredisе®№еҷЁеҶ…жҳҜдёҚеёҰй…ҚзҪ®ж–Үд»¶зҡ„

зӣҙжҺҘжҢӮиҪҪж–Үд»¶еӨ№зҡ„иҜқпјҢеҰӮжһңе®ҝдё»жңәеҶ…дёәз©әж–Үд»¶еӨ№пјҢйӮЈд№Ҳе®№еҷЁеҶ…жҢӮиҪҪзҡ„ж–Үд»¶еӨ№йҮҢзҡ„еҶ…е®№е°Ҷдјҡиў«жё…з©ә
еҰӮжһңе®ҝдё»жңәеҶ…дёәйқһз©әж–Үд»¶еӨ№зҡ„иҜқпјҢйӮЈд№Ҳе®№еҷЁеҶ…жҢӮиҪҪзҡ„ж–Үд»¶еӨ№йҮҢзҡ„еҶ…е®№е°Ҷдјҡиў«жё…з©әпјҢеҶҚе°Ҷе®ҝдё»жңәеҶ…зҡ„ж–Үд»¶еӨҚеҲ¶еҲ°е®№еҷЁеҶ…зҡ„ж–Үд»¶еӨ№йҮҢ

зӣҙжҺҘжҢӮиҪҪж–Үд»¶зҡ„иҜқпјҲdockerеҸҜд»ҘзӣҙжҺҘжҢӮиҪҪж–Үд»¶пјүпјҢз”Ёvimдҝ®ж”№е®ҝдё»жңәж–Үд»¶еҗҺпјҢеҸӘжңүйҮҚеҗҜе®№еҷЁеҗҺпјҢе®№еҷЁеҶ…ж–Үд»¶еҶ…е®№жүҚдјҡеҸҳеҢ–гҖӮиҰҒжғідёҚйҮҚеҗҜпјҢйңҖиҰҒеҜ№е®№еҷЁеҶ…зҡ„ж–Үд»¶иөӢдәҲ777жқғйҷҗпјҢchmod 777 redis.confгҖӮжүҖд»ҘжҢӮиҪҪж–Үд»¶еӨ№дјҡж–№дҫҝдәӣгҖӮ

жҢӮиҪҪеҗҺпјҢж”№еҸҳе®ҝдё»жңәж–Үд»¶пјҢе®№еҷЁеҶ…ж–Үд»¶дјҡеҸҳеҢ–пјӣж”№еҸҳе®№еҷЁеҶ…ж–Үд»¶пјҢе®ҝдё»жңәеҶ…ж–Үд»¶еҗҢж ·дјҡеҸҳеҢ–

(dockerеңЁиҝӣиЎҢж–Үд»¶жҢӮиҪҪж—¶пјҢе№¶дёҚжҳҜд»…д»…жҢӮиҪҪж–Үд»¶еҗҚеҲ°еҜ№еә”дҪҚзҪ®пјҢиҖҢжҳҜе°Ҷж–Үд»¶еҜ№еә”зҡ„inode иҝӣиЎҢжҳ е°„гҖӮз”ЁvimиҝӣиЎҢж–Үд»¶зҡ„зј–иҫ‘е№¶дҝқеӯҳж—¶пјҢзі»з»ҹйҮҮз”Ёзҡ„жҳҜеӨҮд»ҪгҖҒжӣҝжҚўзҡ„зӯ–з•ҘпјҢж–Үд»¶з”Ёvimзӯүе·Ҙе…·зј–иҫ‘зҡ„иҝҮзЁӢе®һиҙЁжҳҜпјҢеӨҮд»ҪеҺҹжқҘзҡ„ж–Үд»¶пјҢеҪ“ж–°ж–Үд»¶зј–иҫ‘е®ҢжҲҗеҗҺпјҢеҶҚе°Ҷж–°ж–Үд»¶жӣҝжҚўж–ҮеҺҹд»¶пјҢиҝҷдјҡеҜјиҮҙж–Үд»¶зҡ„inodeеҸҳеҢ–пјҢжүҖд»ҘdockerеҶ…еӨ–зҡ„ж–Үд»¶е№¶дёҚдјҡеҗҢжӯҘгҖӮиҖҢз”ЁcatпјҢechoзӯүйҮҚе®ҡеҗ‘ж“ҚдҪңдҝ®ж”№ж–Үд»¶ж—¶пјҢж–Үд»¶зҡ„inodeдҝқжҢҒдёҚеҸҳпјҢжүҖд»ҘдёҚдјҡеҸ‘з”ҹзұ»дјјзҺ°иұЎгҖӮ)

nginxй…ҚзҪ®ж–Үд»¶

й…ҚзҪ®ж–Үд»¶з»“жһ„

  • е…ЁеұҖеқ—пјҡй…ҚзҪ®еҪұе“Қnginxе…ЁеұҖзҡ„жҢҮд»ӨгҖӮдёҖиҲ¬жңүиҝҗиЎҢnginxжңҚеҠЎеҷЁзҡ„з”ЁжҲ·з»„пјҢnginxиҝӣзЁӢpidеӯҳж”ҫи·Ҝеҫ„пјҢж—Ҙеҝ—еӯҳж”ҫи·Ҝеҫ„пјҢй…ҚзҪ®ж–Үд»¶еј•е…ҘпјҢе…Ғи®ёз”ҹжҲҗworker processж•°зӯүгҖӮ
  • eventsеқ—пјҡй…ҚзҪ®еҪұе“ҚnginxжңҚеҠЎеҷЁжҲ–дёҺз”ЁжҲ·зҡ„зҪ‘з»ңиҝһжҺҘгҖӮжңүжҜҸдёӘиҝӣзЁӢзҡ„жңҖеӨ§иҝһжҺҘж•°пјҢйҖүеҸ–е“Әз§ҚдәӢд»¶й©ұеҠЁжЁЎеһӢеӨ„зҗҶиҝһжҺҘиҜ·жұӮпјҢжҳҜеҗҰе…Ғи®ёеҗҢж—¶жҺҘеҸ—еӨҡдёӘзҪ‘и·ҜиҝһжҺҘпјҢејҖеҗҜеӨҡдёӘзҪ‘з»ңиҝһжҺҘеәҸеҲ—еҢ–зӯүгҖӮ
  • httpеқ—пјҡз”ЁдәҺй…ҚзҪ®Nginxзҡ„дёҡеҠЎеҠҹиғҪпјҢеҸҜд»ҘеөҢеҘ—еӨҡдёӘserverпјҢй…ҚзҪ®д»ЈзҗҶпјҢзј“еӯҳпјҢж—Ҙеҝ—е®ҡд№үзӯүз»қеӨ§еӨҡж•°еҠҹиғҪе’Ңз¬¬дёүж–№жЁЎеқ—зҡ„й…ҚзҪ®гҖӮеҰӮж–Үд»¶еј•е…ҘпјҢmime-typeе®ҡд№үпјҢж—Ҙеҝ—иҮӘе®ҡд№үпјҢжҳҜеҗҰдҪҝз”Ёsendfileдј иҫ“ж–Үд»¶пјҢиҝһжҺҘи¶…ж—¶ж—¶й—ҙпјҢеҚ•иҝһжҺҘиҜ·жұӮж•°зӯүгҖӮ
  • serverеқ—пјҡй…ҚзҪ®иҷҡжӢҹдё»жңәзҡ„зӣёе…іеҸӮж•°пјҢдёҖдёӘhttpдёӯеҸҜд»ҘжңүеӨҡдёӘserverгҖӮеҝ…йЎ»дҪҚдәҺhttpеҶ…йғЁпјҢз”ЁдәҺй…ҚзҪ®Nginxзҡ„дёҖдёӘдё»жңәпјӣе…¶дёӯlistenжҢҮе®ҡзӣ‘еҗ¬зҡ„з«ҜеҸЈеҸ·пјӣserver_nameжҢҮе®ҡдё»жңәеҗҚпјӣlocationз”ЁдәҺжҢҮе®ҡдё»жңәдёҠзҡ„иө„жәҗдҪҚзҪ®гҖӮ
  • locationеқ—пјҡй…ҚзҪ®иҜ·жұӮзҡ„и·Ҝз”ұпјҢд»ҘеҸҠеҗ„з§ҚйЎөйқўзҡ„еӨ„зҗҶжғ…еҶөгҖӮ
